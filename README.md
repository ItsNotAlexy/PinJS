# PinJS
A way to get images from pinterest using puppeteer!

## How to use
- Fork the repo!

- Create a file!

- Add the loader!

```js
const { GetImages } = require('./PinJS/loader')

GetImages('Cats', true);
```

- Run the file! 
`node index.js`
